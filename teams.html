The Blue Alliance (TBA) - Teams

A Tableau Web Data Connector (WDC)

This WDC connects to the TBA Read API, downloads all teams as JSON data, and sends the data to Tableau.

Required Parameter - API Access Key

Optional Parameters

Instructions

Access Key

This Web Data Connector requires that you have your own TBA API key. To get an API key:

  1. Go to https://www.thebluealliance.com/mytba
  2. Log in or create an account.
  3. Click on Account in the More menu.
  4. Scroll down to Read API Keys
  5. Follow on-screen instructions to create a new API key.

Optional Parameters

If these parameters are left blank, all FRC teams will be retrieved.

  • If a district key is included, e.g., "2020pnw", only teams from the specified district and year will be retrieved.
  • If a year is included, only teams that competed during the specified year will be retrieved. Use a four-digit year. This field is ignored if a district is provided.
